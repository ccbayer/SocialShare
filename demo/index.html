Social Share Demo

Twitter

Parameters for Twitter:

  • url - URL to be tweeted / shared.
  • text - Tweet text
  • hashtags - comma-separated list of hashtags to include
  • via - Twitter handle for attribution

JavaScript Usage

Share on Twitter

Markup looks like this:

<a href="javascript:void(0);" class="share" id="twitter">Share on Twitter</a>

JS looks like this:

$("#twitter").on("click", function() { SocialShare.share("twitter", { url: "share-page.html", text: "Check out this website", hashtags: "SocialShareTool, Tooling", via: "mlom" }); });

HTML Attribute Usage

Share on Twitter

Markup looks like this:

<div data-share-text="title value" data-share-hashtags="hashtag1, hashtag2" class="share-twitter">Share on Twitter</div>

Pinterest

Parameters for Pintrest:

  • url - URL to be tweeted / shared.
  • media - Media URL.
  • description - description of Pin. Can include #hashtags directly in the description.

JavaScript Usage


Pin this Image

Markup looks like this:

<a href="javascript:void(0);" class="share" id="pinterest">Pin this Image</a>

JS looks like this:

$("#pinterest").on("click", function() { SocialShare.share("pinterest", { url: "share-page.html", mediaurl: "http://nicenicejpg.com/600/400", description: "Check out this #vanilla #ice #photo" }); });

HTML Attribute Usage


Pin this Other Image

Markup looks like this:

<div data-share-text="title value" data-share-hashtags="hashtag1, hashtag2" class="share-twitter">Share on Twitter</div>
